> Add basic authentication to requests.

## Installation

```
npm install popsicle-basic-auth --save
```

## Usage

```javascript
var request = require('popsicle')
var auth = require('popsicle-basic-auth')

request('/users.json')
  .use(auth('blakeembrey', 'hunter2'))
```
